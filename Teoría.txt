Talking about XML
XPATH
    -NodeName: se utiliza para seleccionar todos los nodos con el nombre indicado
    -Root(/): se utiliza para seleccionar el elemento raíz con el nombre especificado.
    -Node(//): este comando es para seleccionar todos los nodos con el nombre indicado.
    -Atributo (@): Se utiliza para seleccionar todos los atributos con el nombre indicado.
